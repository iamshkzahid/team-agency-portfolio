About Us

We are a team of OJT trainees working together to build a suite of six web applications. This capstone project demonstrates our understanding of frontend development fundamentals including DOM manipulation, API integration, state management, and responsive design.

Every project is built from scratch using only HTML5, CSS3, and Vanilla JavaScript — no frameworks, no libraries. Each application solves a real-world problem and showcases different aspects of modern web development.
